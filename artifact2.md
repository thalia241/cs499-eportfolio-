### Narrative

The artifact selected for this enhancement is a Python-based text adventure game titled Maze Adventure Game. The original version of the project was created during previous coursework to demonstrate foundational programming concepts including dictionaries, lists, loops, functions, conditional statements, and user interaction. The game allows players to navigate through a maze, collect items, and defeat the Minotaur after gathering the required inventory items. While the original project successfully met the original course requirements, the code review process identified several opportunities for improvement involving data organization, gameplay logic, inventory management, and overall program structure. The enhancement process focused on improving the application’s algorithms, data structures, modularity, and gameplay functionality to better reflect professional programming practices and stronger software design principles.

Additional enhancements included implementing inventory search functionality, item requirement validation, gameplay statistics, score tracking, move tracking, and visited room tracking. These changes increased the complexity of the application while demonstrating stronger algorithmic processing and state management techniques.
